# 使用手册

## 基本使用
1. 打开首页，输入共享密钥（自动保存在浏览器）
2. 输入文本后按 Enter 发送（Shift+Enter 换行）
3. 其他设备输入同一密钥即可实时解密消息

## 发送图片
- **粘贴**：在输入框中粘贴截图/图片即可发送
- **上传**：点击“图片”按钮选择本地图片发送

## 本地记录
- 仅保存文本历史，图片不进入本地记录
- 点击“清空本地”可清除当前密钥的文本记录

## 限制说明
- 图片默认限制 512KB（可在代码中修改）
- 不同网络可用，只要能访问网页即可加入

## 隐私说明
- Worker 只转发，不存储任何消息或图片
- 加密密钥保存在浏览器本地，服务端不可见
